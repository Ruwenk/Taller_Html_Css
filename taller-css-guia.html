ACTIVIDAD 01
🎨 Conecta y colorea
⏱ 45 minutos

Vamos a hacer que el CSS sea lo primero que el navegador aplique. Primero conectamos, luego pintamos el fondo, luego el texto del h1. Cada paso es corto: escribe, guarda, mira el cambio.

1️⃣
Abre style.css — empieza con el comentario de sección
Los comentarios te ayudan a organizar tu CSS. Siempre úsalos.
/* ═══════════════════════════
   ACTIVIDAD 1 — Colores base
═══════════════════════════ */
💡 ¿Por qué? El comentario no hace nada visual, pero organiza tu código para que tú (y otros) puedan entenderlo después.
2️⃣
Dale fondo oscuro al body
El selector body aplica a toda la página
body {
  background-color: #0F172A;  /* azul muy oscuro */
  color: #E2E8F0;           /* texto claro */
}
⚡ Prueba esto: Guarda y mira. ¿La página cambió a fondo oscuro? Si no cambió, verifica que el archivo CSS está en la misma carpeta que el HTML y que el link está correcto.

3️⃣
Colorea el h1 con el morado especial
Selector de etiqueta: aplica a TODOS los h1 de la página
h1 {
  color: #8B5CF6;  /* morado — el color de CSS Mundo 03 */
}
💡 Diferencia clave: color cambia el texto. background-color cambia el fondo. Son dos propiedades distintas.
4️⃣
Experimenta con formatos de color
CSS acepta colores en 4 formatos distintos. Pruébalos.
FORMATOEJEMPLOCUÁNDO USARLO
#HEX#8B5CF6El más común. Cópialo del diseño.
nombrepurple, red, bluePara pruebas rápidas. ~140 disponibles.

/* Prueba cambiar el color del h2 con cada formato */
h2 {
  color: purple;          /* nombre */
  /* color: #9333EA; */      /* hex */
  /* color: rgb(147,51,234); */  /* rgb */
}
🎯 Reto de exploración: Cambia el color del h2 con los 4 formatos distintos, uno a la vez. Guarda y mira. ¿El resultado visual es el mismo? ¿Por qué crees que hay 4 formatos si todos dan color?
✅ Al terminar la Actividad 1 debes tener: Fondo oscuro en toda la página, texto principal claro, h1 morado, h2 con color. El CSS tiene exactamente 3–4 reglas.
ACTIVIDAD 02
✍️ El Tipógrafo
⏱ 45 minutos

Las fuentes le dan personalidad al diseño. Vamos a cambiar tipografía, tamaños, grosores, alineación y espaciado de texto. Al final tu página se verá completamente diferente solo con propiedades de texto.

1️⃣
Cambia la fuente de toda la página
font-family en el body se hereda a todos los hijos
body {
  background-color: #0F172A;
  color: #E2E8F0;
  font-family: 'Segoe UI', Arial, sans-serif; /* ← agrega esto */
}
💡 ¿Por qué tres valores? Si el navegador no tiene 'Segoe UI', prueba Arial. Si tampoco lo tiene, usa cualquier fuente sin serif. Es un plan de respaldo.
FUENTEESTILO
'Segoe UI', sans-serifModerna, limpia — Windows/Android
Georgia, serifElegante, editorial
Consolas, monospaceCódigo, técnico, developer
Impact, sans-serifBold, impactante, posters
2️⃣
Ajusta el tamaño del h1 y el espacio entre líneas
font-size y line-height son los más importantes para legibilidad
h1 {
  color: #8B5CF6;
  font-size: 3rem;      /* 3rem = 48px (1rem = 16px por defecto) */
  font-weight: 900;    /* 900 = black (el más grueso) */
  line-height: 1.1;   /* espacio entre líneas — 1.1 es compacto */
}

// font-weight: grosores

3️⃣
Propiedades de texto: align, decoration y transform
Controlan la alineación, subrayado y mayúsculas
h2 {
  color: #A78BFA;            /* morado claro */
  font-size: 1.5rem;
  text-transform: uppercase;  /* TODO EN MAYÚSCULAS */
  letter-spacing: 0.1em;     /* espacio entre letras */
}

a {
  color: #22C55E;             /* verde */
  text-decoration: none;     /* quita el subrayado de los links */
}

p {
  line-height: 1.7;          /* más espacio = más fácil de leer */
  color: #94A3B8;            /* gris claro para párrafos */
}
🎯 Experimenta: Prueba text-align: center en el h1. Luego prueba right y justify. ¿Cuál se ve mejor para un título de perfil?
4️⃣
Agrega line-height al body para mejor lectura
Un line-height de 1.6–1.7 hace los párrafos mucho más legibles
body {
  background-color: #0F172A;
  color: #E2E8F0;
  font-family: 'Segoe UI', Arial, sans-serif;
  line-height: 1.6;      /* ← esto mejora la legibilidad global */
  font-size: 1rem;        /* tamaño base del texto normal */
}
✅ Al terminar la Actividad 2 debes tener: Fuente cambiada en toda la página, h1 grande y bold, h2 en uppercase con espaciado, links sin subrayado, párrafos con buen interlineado. Tu CSS ya tiene unas 8–10 reglas.
ACTIVIDAD 03
🎯 Clases y Selectores
⏱ 60 minutos

Hasta ahora usamos selectores de etiqueta (h1, p, body). Ahora vamos a usar clases y IDs — estos nos permiten estilizar elementos específicos sin afectar a todos los demás.

1️⃣
Estiliza la clase .nombre y .rol
El punto (.) indica que es una clase — aplica solo a los elementos con ese class
/* ═══════════════════════════
   ACTIVIDAD 3 — Clases e IDs
═══════════════════════════ */

.nombre {
  color: #8B5CF6;
  font-size: 2.5rem;
  font-weight: 800;
  text-transform: uppercase;
  letter-spacing: 0.05em;
}

.rol {
  color: #94A3B8;
  font-size: 1.1rem;
  font-style: italic;   /* cursiva para el subtítulo */
}
💡 Clase vs etiqueta: h1 { } afecta TODOS los h1 de la página. .nombre { } solo afecta los elementos que tengan class="nombre". Puedes mezclarlos libremente.
2️⃣
Estiliza .descripcion y .boton
El botón necesita background-color — aquí aparece por primera vez en el proyecto
.descripcion {
  color: #CBD5E1;
  font-size: 1.1rem;
  line-height: 1.8;
  max-width: 500px;     /* no dejar que el texto sea demasiado ancho */
}

.boton {
  background-color: #8B5CF6;  /* fondo morado */
  color: white;                /* texto blanco */
  text-decoration: none;
  font-weight: 700;
  letter-spacing: 0.05em;
  display: inline-block;    /* necesario para aplicar padding */
  padding: 0.75rem 1.5rem;   /* espacio interno (lo veremos en Mundo 04) */
  border-radius: 8px;        /* esquinas redondeadas */
}
🎯 Reto de personalización: Cambia el color del botón a #22C55E (verde). ¿Se ve diferente? ¿Cuál prefieres? Ahora vuelve al morado.
3️⃣
Estiliza las .tag (las bolitas de habilidades)
Una clase → muchos elementos — las 4 habilidades se estilizan con UNA regla
.tag {
  background-color: rgba(139, 92, 246, 0.15); /* morado transparente */
  color: #A78BFA;
  font-size: 0.85rem;
  font-weight: 700;
  letter-spacing: 0.08em;
  text-transform: uppercase;
  display: inline-block;
  padding: 0.3rem 0.8rem;
  border-radius: 100px;      /* muy redondeado = "pill" */
}
💡 Magia del rgba: rgba(139, 92, 246, 0.15) es el mismo morado pero con solo un 15% de opacidad. El último número (0 a 1) controla la transparencia. Prueba con 0.5 y con 1 para ver la diferencia.
4️⃣
Usa el selector de ID para la sección #contacto
El # indica que es un ID — solo puede haber uno en la página
#contacto {
  background-color: #1E293B;  /* fondo diferente para esta sección */
  text-align: center;
  padding: 3rem 2rem;
}

.email {
  color: #22C55E;
  font-size: 1.2rem;
  font-weight: 700;
  letter-spacing: 0.05em;
}
SELECTORSÍMBOLOCUÁNDO USARLO
etiqueta(ninguno)Todos los h1, p, body...
.clase.Grupos de elementos similares
#id#UN elemento único (logo, sección especial)
**Absolutamente todos los elementos
✅ Al terminar la Actividad 3 debes tener: Nombre grande y morado, botón con fondo de color, tags/píldoras de habilidades estilizadas, sección de contacto con fondo diferente. Estás usando selectores de etiqueta, clase e ID.
ACTIVIDAD 04
🏗️ Secciones bonitas
⏱ 60 minutos

Ahora vamos a trabajar con las partes estructurales: el header (nav), las tarjetas de proyectos y el footer. Aquí la página empieza a verse como un sitio real.

1️⃣
Estiliza el header y el nav
El header tiene fondo propio y el nav organiza los links
/* ═══════════════════════════
   ACTIVIDAD 4 — Secciones
═══════════════════════════ */

header {
  background-color: #1E293B;  /* un poco más claro que el body */
  padding: 1rem 2rem;
}

nav a {                          /* selector descendiente: links DENTRO del nav */
  color: #94A3B8;
  text-decoration: none;
  font-weight: 600;
  font-size: 0.9rem;
  letter-spacing: 0.05em;
  text-transform: uppercase;
  margin-right: 1.5rem;         /* espacio entre los links del nav */
}
💡 Selector descendiente: nav a significa "los elementos <a> que estén DENTRO de un <nav>". No afecta a los links que están en el main o el footer. Muy útil para estilos específicos.
2️⃣
Estiliza las secciones principales
Cada section necesita su propio respiro (padding) y ancho máximo
section {
  padding: 4rem 2rem;
  max-width: 800px;
  margin: 0 auto;           /* centrar el contenido */
}

#inicio {
  text-align: center;
  padding: 5rem 2rem;       /* la sección hero tiene más padding */
}

#habilidades {
  background-color: #1E293B;
  max-width: 100%;            /* esta sección va a todo el ancho */
  text-align: center;
}
3️⃣
Las tarjetas de proyectos
Ahora usamos .tarjeta y .tarjeta.destacada — dos clases a la vez
.tarjeta {
  background-color: #1E293B;
  border-radius: 12px;
  padding: 1.5rem;
  margin-bottom: 1rem;
}

.tarjeta h3 {
  color: #E2E8F0;
  font-size: 1.1rem;
  margin-bottom: 0.5rem;
}

/* Tarjeta especial: tiene las dos clases "tarjeta destacada" */
.tarjeta.destacada {
  background-color: rgba(139, 92, 246, 0.1);
  border: 1px solid rgba(139, 92, 246, 0.4);  /* borde morado */
}

.link-proyecto {
  color: #22C55E;
  text-decoration: none;
  font-weight: 700;
  font-size: 0.9rem;
}
💡 Dos clases: El HTML dice class="tarjeta destacada" — tiene DOS clases. En CSS, .tarjeta.destacada (sin espacio entre los puntos) selecciona solo los elementos que tengan AMBAS clases. Con espacio sería un selector descendiente.
4️⃣
El footer — cierre del documento
Oscuro, centrado, pequeño — el footer es el cierre tranquilo de la página
footer {
  background-color: #0a0f1a;  /* más oscuro que el body */
  color: #475569;              /* gris oscuro */
  text-align: center;
  padding: 2rem;
  font-size: 0.85rem;
  letter-spacing: 0.05em;
}
✅ Al terminar la Actividad 4 debes tener: Nav estilizado en el header, tarjetas con fondo diferente al body, la tarjeta "destacada" con borde morado, footer con cierre visual. La página ya se ve como un sitio real.

Agrega la fuente de Google Fonts
Google Fonts es gratis y tiene más de 1.000 fuentes — dale personalidad a tu página

Ve a fonts.google.com, elige una fuente que te guste, copia el link y pégalo en el <head> de tu HTML, ANTES del link de tu style.css:

<!-- Pega aquí el link de Google Fonts -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;700;900&display=swap" rel="stylesheet">
<link rel="stylesheet" href="style.css">  <!-- el tuyo va de último -->
/* En el CSS, úsala así: */
body {
  font-family: 'Poppins', sans-serif;  /* el nombre exacto que diga en Google Fonts */
}

4️⃣
Revisa tu CSS completo — checklist final
Antes de presentar, verifica que hayas usado todas las propiedades del taller

// Propiedades usadas

  • color — texto de algún elemento
  • background-color — fondo de al menos 2 elementos
  • font-family — fuente del body
  • font-size — al menos 3 tamaños distintos
  • font-weight — bold en algún elemento
  • line-height — en body o párrafos
  • text-align — al menos una vez
  • text-decoration — en los links
  • text-transform — en h2 o las tags
  • letter-spacing — en algún título

// Selectores usados

  • Selector de etiqueta: body, h1, p...
  • Selector de clase: .nombre, .tarjeta...
  • Selector de ID: #contacto u otro
  • Selector descendiente: nav a
  • Dos clases: .tarjeta.destacada

// Formatos de color usados

  • Hexadecimal: #8B5CF6
  • Nombre: white
  • rgba(): rgba(139,92,246,0.15)
